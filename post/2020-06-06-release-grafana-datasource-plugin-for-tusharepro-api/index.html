发布基于tushare.pro的api的grafana datasource插件
grafana 炒股 grafana plugin datasource stock 炒股
Lastmod: 2020-06-06

很多炒股软件的图形化界面不易使用，特别是想交叉或者复杂分析时，力不从心。grafana是一个非常好的数据看板，而且还在快速发展中。所以有了将tushare.pro的数据在grafana中看的想法。

在grafana中，添加数据源需要开发datasource插件。感谢下grafana-finance，没有它提供的思路，作为一个后端开发的我很难短时间完成。

由于我的tushare.pro的账号的积分不多，目前仅提供daily日线的数据源，今后可以加入更多数据类型的数据源。

依赖

grafana >= 3.0.0即可，也支持最新的grafana 7.x

安装

去release页面下载最新版本，解压到grafana的plugins目录，重启启动grafana即可。

配置

关于CORS跨域的说明

由于此datasource是grafana的普通datasouce，请求都是从浏览器直接发出的，所以需要后端数据api支持CORS跨域，目前tushare.pro的api应该还不支持。目前为了使用此插件，需要使用跨域代理，可以使用cors-anywhere
